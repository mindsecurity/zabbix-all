## RocketChat Script for Zabbix

* **VARIABLES**
```
URL         - Full URL of Webhook>
LOGFILE     - Full Path of Log File (eg. /var/log/zabbix/zabbix-rocketChat.log)

TO          - Chanel or User for Zabbix Conf.
SUBJECT     - The Subject of Message
MESSAGE     - The Message
```
This will be part of the message on rocketchat:
```
DASHBOARD   - Full URL of Zabbix Dashboard
COLOR       - The Color of RocketChat Notification
EMOJI       - The Emoji Used on RocketChat Notification
PAYLOAD     - The JSON Content of The RocketChat Message
```
